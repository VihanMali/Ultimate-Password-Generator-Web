# Ultimate Password Generator - Web Version

This is the **web version** of my Ultimate Password Generator project, originally implemented in Java.  
It allows users to generate strong, secure passwords directly in the browser using HTML, CSS, and JavaScript.

---

## Features

- Select password strength: **Easy**, **Hard**, **Ultimate**  
- Dynamic inputs appear based on selected strength  
- Generates passwords with:
  - Multiple words
  - Random special characters
  - Reversed or alternating case letters for maximum security  
- Fully client-side, no backend required  

---

## How to Use

1. Open `index.html` in your browser  
2. Enter your password words in the input fields  
3. Select the desired password strength from the dropdown  
4. Click **Generate**  
5. Your password will appear below the button
